# ClipFlow Beta

## Highlights

- dense main window and tray workflow built for fast clipboard recall
- settings-first welcome and release notes instead of blocking the main history view
- snippets, diagnostics, export, recovery, and local-first trust surfaces
- release metadata, checksums, and validation artifacts included with beta drops

## Current Trust Status

This beta may still be in one of two trust states:

- fully signed and notarized
- structurally valid but ad-hoc signed while Apple signing setup is still in progress

If the current beta is not notarized yet, use the install guide for the local-open workaround:

- [docs/install-and-upgrade.md](../install-and-upgrade.md)
